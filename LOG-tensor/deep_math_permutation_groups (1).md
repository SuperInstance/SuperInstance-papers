## 1. The Symmetric Group S_n

### 1.1 Definition and Basic Structure

The **symmetric group** $S_n$ is the group of all bijections (permutations) of the set $\{1, 2, \ldots, n\}$ under composition.

**Definition 1.1.1** (Symmetric Group):
$$S_n = \{\sigma : [n] \to [n] \mid \sigma \text{ is a bijection}\}$$

with group operation:
$$(\sigma \tau)(i) = \sigma(\tau(i))$$

**Theorem 1.1.2** (Order of S_n):
$$|S_n| = n!$$

*Proof*: For $\sigma(1)$ we have $n$ choices, for $\sigma(2)$ we have $n-1$ remaining choices, etc. Thus:
$$|S_n| = n \cdot (n-1) \cdot (n-2) \cdots 2 \cdot 1 = n! \quad \square$$

### 1.2 Generators and Relations

**Definition 1.2.1** (Transposition):
A **transposition** $(i\ j)$ is a permutation that swaps $i$ and $j$ and fixes all other elements:
$$(i\ j)(k) = \begin{cases} j & k = i \\ i & k = j \\ k & \text{otherwise} \end{cases}$$

**Theorem 1.2.2** (Generating Set):
$S_n$ is generated by adjacent transpositions:
$$S_n = \langle s_1, s_2, \ldots, s_{n-1} \rangle$$
where $s_i = (i\ i+1)$.

*Proof Sketch*: Any permutation can be written as a product of transpositions. Any transposition $(i\ j)$ can be written as:
$$(i\ j) = s_i s_{i+1} \cdots s_{j-2} s_{j-1} s_{j-2} \cdots s_{i+1} s_i \quad \square$$

**Theorem 1.2.3** (Coxeter Presentation):
$S_n$ has the Coxeter presentation with generators $\{s_1, \ldots, s_{n-1}\}$ and relations:
$$\begin{aligned}
s_i^2 &= e && \text{(involution)} \\
s_i s_j &= s_j s_i && \text{if } |i - j| > 1 \text{ (far commutativity)} \\
s_i s_{i+1} s_i &= s_{i+1} s_i s_{i+1} && \text{(braid relation)}
\end{aligned}$$

### 1.3 Cycle Decomposition

**Definition 1.3.1** (Cycle):
A **k-cycle** $(a_1\ a_2\ \cdots\ a_k)$ is the permutation:
$$a_1 \mapsto a_2 \mapsto a_3 \mapsto \cdots \mapsto a_k \mapsto a_1$$

**Theorem 1.3.2** (Cycle Decomposition):
Every $\sigma \in S_n$ can be uniquely expressed as a product of disjoint cycles (up to order and cyclic rotation within cycles).

**Definition 1.3.3** (Cycle Type):
The **cycle type** of $\sigma$ is the partition $\lambda \vdash n$ given by the lengths of cycles in the decomposition.

**Example 1.3.4**:
For $n = 5$, $\sigma = (1\ 3)(2\ 4\ 5)$ has cycle type $(3, 2)$ or equivalently $\lambda = (3, 2) \vdash 5$.

### 1.4 Conjugacy Classes

**Theorem 1.4.1** (Conjugacy Class Classification):
Two permutations $\sigma, \tau \in S_n$ are conjugate if and only if they have the same cycle type.

*Proof*: 
- ($\Rightarrow$) If $\tau = \pi\sigma\pi^{-1}$ and $\sigma$ has cycle $(a_1\ \cdots\ a_k)$, then $\tau$ has cycle $(\pi(a_1)\ \cdots\ \pi(a_k))$.
- ($\Leftarrow$) If $\sigma, \tau$ have the same cycle type, construct $\pi$ mapping corresponding cycle elements.

**Corollary 1.4.2** (Number of Conjugacy Classes):
The number of conjugacy classes of $S_n$ equals the number of partitions of $n$, denoted $p(n)$.

**Asymptotic Formula** (Hardy-Ramanujan):
$$p(n) \sim \frac{1}{4n\sqrt{3}} \exp\left(\pi\sqrt{\frac{2n}{3}}\right)$$

### 1.5 Young Diagrams and Tableaux

**Definition 1.5.1** (Young Diagram):
A **Young diagram** of a partition $\lambda = (\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_k)$ is a left-justified array of boxes with $\lambda_i$ boxes in row $i$.

**Example 1.5.2**:
For $\lambda = (4, 2, 1)$:
```
┌───┬───┬───┬───┐
│   │   │   │   │
├───┼───┼───┼───┘
│   │   │
├───┼───┤
│   │
└───┘
```

**Definition 1.5.3** (Young Tableau):
A **Young tableau** of shape $\lambda$ is a filling of the boxes with numbers $1, 2, \ldots, n$ (bijectively).

**Definition 1.5.4** (Standard Young Tableau - SYT):
A SYT is a Young tableau where entries increase left-to-right in rows and top-to-bottom in columns.

**Theorem 1.5.5** (Hook Length Formula):
The number of SYT of shape $\lambda$ is:
$$f^\lambda = \frac{n!}{\prod_{(i,j) \in \lambda} h(i,j)}$$

where $h(i,j)$ is the **hook length** at position $(i,j)$:
$$h(i,j) = \lambda_i - j + \lambda'_j - i + 1$$

with $\lambda'_j$ being the conjugate partition (transpose of diagram).

**Example 1.5.6**:
For $\lambda = (3, 2)$, the hook lengths are:
```
┌───┬───┬───┐
│ 4 │ 3 │ 1 │
├───┼───┼───┤
│ 2 │ 1 │
└───┴───┘
```

Thus:
$$f^{(3,2)} = \frac{5!}{4 \cdot 3 \cdot 1 \cdot 2 \cdot 1} = \frac{120}{24} = 5$$

## 2. Permutation Representations

### 2.1 Permutation Matrices

**Definition 2.1.1** (Permutation Matrix):
For $\sigma \in S_n$, the **permutation matrix** $P_\sigma \in \text{GL}(n, \mathbb{C})$ is:
$$(P_\sigma)_{ij} = \delta_{i, \sigma(j)} = \begin{cases} 1 & \text{if } i = \sigma(j) \\ 0 & \text{otherwise} \end{cases}$$

**Properties**:
1. $P_\sigma P_\tau = P_{\sigma\tau}$ (homomorphism)
2. $P_\sigma^{-1} = P_{\sigma^{-1}} = P_\sigma^T$
3. $\det(P_\sigma) = \text{sgn}(\sigma) = (-1)^{\text{inv}(\sigma)}$

**Proposition 2.1.2** (Matrix Action):
For $v \in \mathbb{C}^n$:
$$P_\sigma v = (v_{\sigma^{-1}(1)}, v_{\sigma^{-1}(2)}, \ldots, v_{\sigma^{-1}(n)})^T$$

### 2.2 Regular Representation

**Definition 2.2.1** (Group Algebra):
The **group algebra** $\mathbb{C}[S_n]$ is the vector space with basis $\{e_\sigma : \sigma \in S_n\}$ and multiplication:
$$e_\sigma \cdot e_\tau = e_{\sigma\tau}$$

**Definition 2.2.2** (Regular Representation):
The **(left) regular representation** $\rho_{\text{reg}}$ acts on $\mathbb{C}[S_n]$ by:
$$\rho_{\text{reg}}(\sigma)\left(\sum_{\tau} c_\tau e_\tau\right) = \sum_{\tau} c_\tau e_{\sigma\tau}$$

**Theorem 2.2.3** (Regular Representation Decomposition):
$$\rho_{\text{reg}} \cong \bigoplus_{\lambda \vdash n} (f^\lambda) V^\lambda$$

where $V^\lambda$ is the irreducible representation indexed by $\lambda$ and $f^\lambda$ is its dimension.

### 2.3 Characters

**Definition 2.3.1** (Character):
The **character** of a representation $\rho$ is:
$$\chi_\rho(\sigma) = \text{tr}(\rho(\sigma))$$

**Proposition 2.3.2** (Permutation Matrix Character):
$$\chi_{\text{perm}}(\sigma) = |\{i : \sigma(i) = i\}| = \text{number of fixed points}$$

**Theorem 2.3.3** (Irreducible Characters of S_n):
The irreducible characters $\chi^\lambda$ of $S_n$ are indexed by partitions $\lambda \vdash n$. The **Frobenius formula** computes:
$$\chi^\lambda(\sigma) = \Delta_\lambda(p_1, p_2, \ldots)$$

where $p_k$ are power sums and $\Delta_\lambda$ is a determinant.

**Murnaghan-Nakayama Rule**:
A recursive formula for computing $\chi^\lambda(\sigma)$ when $\sigma$ has cycle type $\mu$:
$$\chi^\lambda(\mu) = \sum_{T} (-1)^{\text{ht}(T)} \chi^{\lambda \setminus T}(\mu \setminus \mu_1)$$

where the sum is over **rim hooks** $T$ of length $\mu_1$ in $\lambda$.

### 2.4 Irreducible Representations

**Theorem 2.4.1** (Specht Modules):
For each partition $\lambda \vdash n$, there exists an irreducible representation $S^\lambda$ (Specht module) of dimension $f^\lambda$.

**Construction via Young Symmetrizers** (see Section 3.4):
$$S^\lambda = \mathbb{C}[S_n] \cdot c_\lambda$$
where $c_\lambda$ is the Young symmetrizer.

**Character Table for S_3**:

| Conjugacy Class | $e$ | $(1\ 2)$ | $(1\ 2\ 3)$ |
|-----------------|-----|----------|-------------|
| Size | 1 | 3 | 2 |
| $\chi^{(3)}$ (trivial) | 1 | 1 | 1 |
| $\chi^{(1,1,1)}$ (sign) | 1 | -1 | 1 |
| $\chi^{(2,1)}$ (standard) | 2 | 0 | -1 |

### 2.5 Decomposition into Irreducibles

**Theorem 2.5.1** (Decomposition of Permutation Representation):
The natural permutation representation of $S_n$ decomposes as:
$$\mathbb{C}^n \cong V^{(n)} \oplus V^{(n-1, 1)}$$

where:
- $V^{(n)}$ is the trivial representation (1D)
- $V^{(n-1, 1)}$ is the **standard representation** ($(n-1)$D)

**Proof**: The subspace spanned by $(1, 1, \ldots, 1)^T$ is invariant (trivial). Its orthogonal complement, the subspace $\{v : \sum_i v_i = 0\}$, gives $V^{(n-1, 1)}$.

## 3. Algebraic Structures

### 3.1 Group Algebra ℂ[S_n]

**Definition 3.1.1**:
The group algebra $\mathbb{C}[S_n]$ consists of formal sums:
$$a = \sum_{\sigma \in S_n} a_\sigma \sigma, \quad a_\sigma \in \mathbb{C}$$

with:
- Addition: $(a + b)_\sigma = a_\sigma + b_\sigma$
- Multiplication: $(ab)_\sigma = \sum_{\tau} a_\tau b_{\tau^{-1}\sigma}$

**Theorem 3.1.2** (Semisimplicity):
$\mathbb{C}[S_n]$ is semisimple (Maschke's theorem applies). It decomposes as:
$$\mathbb{C}[S_n] \cong \bigoplus_{\lambda \vdash n} M_{f^\lambda}(\mathbb{C})$$

where $M_d(\mathbb{C})$ is the algebra of $d \times d$ matrices.

### 3.2 Center of the Group Algebra

**Definition 3.2.1**:
The center $Z(\mathbb{C}[S_n])$ consists of elements commuting with all of $\mathbb{C}[S_n]$.

**Theorem 3.2.2** (Basis of Center):
$Z(\mathbb{C}[S_n])$ has a basis given by **class sums**:
$$K_C = \sum_{\sigma \in C} \sigma$$
where $C$ runs over conjugacy classes of $S_n$.

**Corollary 3.2.3**:
$$\dim Z(\mathbb{C}[S_n]) = \text{number of partitions of } n = p(n)$$

### 3.3 Central Idempotents

**Definition 3.3.1** (Idempotent):
An element $e$ is **idempotent** if $e^2 = e$.

**Definition 3.3.2** (Primitive Central Idempotent):
A primitive central idempotent is a minimal non-zero idempotent in the center.

**Theorem 3.3.3** (Primitive Central Idempotents of S_n):
For each $\lambda \vdash n$, the primitive central idempotent is:
$$e_\lambda = \frac{f^\lambda}{n!} \sum_{\sigma \in S_n} \chi^\lambda(\sigma) \cdot \sigma$$

**Properties**:
1. $e_\lambda e_\mu = \delta_{\lambda\mu} e_\lambda$
2. $\sum_{\lambda \vdash n} e_\lambda = 1$
3. $e_\lambda \mathbb{C}[S_n] \cong M_{f^\lambda}(\mathbb{C})$

### 3.4 Young Symmetrizers

**Definition 3.4.1** (Row and Column Stabilizers):
For a Young tableau $T$ of shape $\lambda$:
- **Row stabilizer**: $R_T = \{ \sigma \in S_n : \sigma \text{ preserves each row of } T \}$
- **Column stabilizer**: $C_T = \{ \sigma \in S_n : \sigma \text{ preserves each column of } T \}$

**Definition 3.4.2** (Young Symmetrizer):
$$c_T = a_T \cdot b_T = \left(\sum_{\sigma \in R_T} \sigma\right) \left(\sum_{\tau \in C_T} \text{sgn}(\tau) \tau\right)$$

where:
- $a_T$ is the **row symmetrizer**
- $b_T$ is the **column antisymmetrizer**

**Theorem 3.4.3** (Specht Module Construction):
The Specht module $S^\lambda$ is:
$$S^\lambda = \mathbb{C}[S_n] \cdot c_T$$

**Proposition 3.4.4** (Idempotent Property):
$$c_T^2 = n_T \cdot c_T$$
for some scalar $n_T \neq 0$.

**Normalized Young Symmetrizer**:
$$\tilde{c}_T = \frac{c_T}{n_T}$$
is a true idempotent: $\tilde{c}_T^2 = \tilde{c}_T$.

### 3.5 Connection to Representations

**Theorem 3.5.1** (Projection onto Isotypic Component):
$e_\lambda$ projects onto the isotypic component of type $\lambda$:
$$V = \bigoplus_{\lambda \vdash n} e_\lambda V$$

**Example 3.5.2** (S_3 Case):
For $S_3$, with $\lambda = (2, 1)$:
$$c_{(2,1)} = (e + (1\ 2))(e - (1\ 3)) = e + (1\ 2) - (1\ 3) - (1\ 2\ 3)$$

This projects onto the 2D standard representation.

## 4. Tensor Operations & Symmetrizers

### 4.1 Tensor Products and Group Actions

**Definition 4.1.1** (Tensor Power):
The $k$-th tensor power of a vector space $V$:
$$V^{\otimes k} = \underbrace{V \otimes V \otimes \cdots \otimes V}_{k \text{ times}}$$

**Definition 4.1.2** (Permutation Action on Tensors):
For $\sigma \in S_k$ and $v_1 \otimes \cdots \otimes v_k \in V^{\otimes k}$:
$$\sigma \cdot (v_1 \otimes \cdots \otimes v_k) = v_{\sigma^{-1}(1)} \otimes \cdots \otimes v_{\sigma^{-1}(k)}$$

**Matrix Form**:
If $P_\sigma$ is the permutation matrix, then on $V^{\otimes k}$:
$$P_\sigma^{\otimes k} = P_\sigma \otimes \cdots \otimes P_\sigma \quad \text{(k factors)}$$

### 4.2 Symmetrizers and Antisymmetrizers

**Definition 4.2.1** (Total Symmetrizer):
$$\mathcal{S} = \frac{1}{k!} \sum_{\sigma \in S_k} \sigma$$

**Properties**:
1. $\mathcal{S}^2 = \mathcal{S}$ (idempotent)
2. $\mathcal{S}(v \otimes w) = \frac{1}{2}(v \otimes w + w \otimes v)$ for $k=2$
3. $\sigma \mathcal{S} = \mathcal{S}$ for all $\sigma \in S_k$

**Definition 4.2.2** (Total Antisymmetrizer):
$$\mathcal{A} = \frac{1}{k!} \sum_{\sigma \in S_k} \text{sgn}(\sigma) \cdot \sigma$$

**Properties**:
1. $\mathcal{A}^2 = \mathcal{A}$
2. $\mathcal{A}(v \otimes w) = \frac{1}{2}(v \otimes w - w \otimes v)$ for $k=2$
3. $\sigma \mathcal{A} = \text{sgn}(\sigma) \mathcal{A}$

**Theorem 4.2.3** (Exterior Power):
$$\Lambda^k V = \mathcal{A}(V^{\otimes k}) = \{v \in V^{\otimes k} : \sigma \cdot v = \text{sgn}(\sigma) v \text{ for all } \sigma\}$$

**Theorem 4.2.4** (Symmetric Power):
$$\text{Sym}^k V = \mathcal{S}(V^{\otimes k}) = \{v \in V^{\otimes k} : \sigma \cdot v = v \text{ for all } \sigma\}$$

### 4.3 Schur Functors

**Definition 4.3.1** (Schur Functor):
For a partition $\lambda = (\lambda_1, \ldots, \lambda_k) \vdash n$, the **Schur functor** $\mathbb{S}_\lambda$ is:
$$\mathbb{S}_\lambda(V) = c_\lambda(V^{\otimes n})$$

where $c_\lambda$ is the Young symmetrizer for a tableau of shape $\lambda$.

**Properties**:
1. $\mathbb{S}_{(n)}(V) = \text{Sym}^n V$
2. $\mathbb{S}_{(1^n)}(V) = \Lambda^n V$
3. $\dim \mathbb{S}_\lambda(\mathbb{C}^d) = s_\lambda(1^d)$ (Schur polynomial)

**Theorem 4.3.2** (Schur-Weyl Duality):
As $(\text{GL}(V), S_n)$-bimodules:
$$V^{\otimes n} \cong \bigoplus_{\lambda \vdash n, \ell(\lambda) \leq \dim V} \mathbb{S}_\lambda(V) \otimes S^\lambda$$

where $S^\lambda$ is the Specht module and $\mathbb{S}_\lambda(V)$ is an irreducible $\text{GL}(V)$-module.

### 4.4 Plethysm

**Definition 4.4.1** (Composition of Functors):
Given Schur functors $\mathbb{S}_\lambda$ and $\mathbb{S}_\mu$, their **plethysm** is:
$$\mathbb{S}_\lambda \circ \mathbb{S}_\mu = \mathbb{S}_\lambda(\mathbb{S}_\mu(-))$$

**Decomposition**:
$$\mathbb{S}_\lambda \circ \mathbb{S}_\mu(V) \cong \bigoplus_\nu c^\nu_{\lambda, \mu} \mathbb{S}_\nu(V)$$

where $c^\nu_{\lambda, \mu}$ are **plethysm coefficients**.

**Example 4.4.2**:
$$\text{Sym}^2(\Lambda^2 V) = \mathbb{S}_{(2)} \circ \mathbb{S}_{(1,1)}(V)$$

decomposes into specific GL(V)-representations.

### 4.5 Application to Tensor Decomposition

**Theorem 4.5.1** (Tensor Product of Representations):
$$\mathbb{S}_\lambda(V) \otimes \mathbb{S}_\mu(V) \cong \bigoplus_\nu c^\nu_{\lambda\mu} \mathbb{S}_\nu(V)$$

where $c^\nu_{\lambda\mu}$ are **Littlewood-Richardson coefficients**.

**Littlewood-Richardson Rule**:
$c^\nu_{\lambda\mu}$ counts the number of Littlewood-Richardson tableaux of shape $\nu/\lambda$ with content $\mu$.

**Example 4.5.2**:
$$\mathbb{S}_{(2,1)} \otimes \mathbb{S}_{(2,1)} = \mathbb{S}_{(4,2)} \oplus \mathbb{S}_{(4,1,1)} \oplus \mathbb{S}_{(3,3)} \oplus 2\mathbb{S}_{(3,2,1)} \oplus \mathbb{S}_{(3,1,1,1)} \oplus \mathbb{S}_{(2,2,2)} \oplus \mathbb{S}_{(2,2,1,1)}$$

## 5. RTT Integration Framework

### 5.1 Permutation Equivariance

**Definition 5.1.1** (Equivariant Function):
A function $f: \mathbb{R}^{n \times d} \to \mathbb{R}^{n \times d'}$ is **permutation equivariant** if:
$$f(P_\sigma X) = P_\sigma f(X)$$
for all $\sigma \in S_n$.

**Definition 5.1.2** (Invariant Function):
A function $f: \mathbb{R}^{n \times d} \to \mathbb{R}^{d'}$ is **permutation invariant** if:
$$f(P_\sigma X) = f(X)$$
for all $\sigma \in S_n$.

### 5.2 Equivariant Architectures

**Theorem 5.2.1** (Universal Approximation for Equivariant Maps):
Any continuous permutation-equivariant map $f: \mathbb{R}^{n \times d} \to \mathbb{R}^{n \times d'}$ can be approximated arbitrarily well by:
$$f(X) = \rho\left(\text{aggregate}(\phi(X))\right)$$

where:
- $\phi: \mathbb{R}^d \to \mathbb{R}^{h}$ is applied pointwise
- aggregate is permutation-invariant (e.g., sum, mean, max)
- $\rho: \mathbb{R}^{h+d} \to \mathbb{R}^{d'}$ is applied pointwise with aggregated features

**Transformer Attention as Equivariant**:
$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

**Proposition 5.2.2**:
Standard multi-head attention is permutation equivariant:
$$\text{Attention}(P_\sigma Q, P_\sigma K, P_\sigma V) = P_\sigma \text{Attention}(Q, K, V)$$

### 5.3 Frame Averaging for Invariant Functions

**Definition 5.3.1** (Frame):
For $X \in \mathbb{R}^{n \times d}$, a **frame** $\mathcal{F}(X) \subseteq S_n$ is a subset of permutations capturing relevant symmetries.

**Definition 5.3.2** (Frame Averaging):
For an arbitrary function $f$, the frame-averaged invariant version is:
$$\bar{f}(X) = \frac{1}{|\mathcal{F}(X)|} \sum_{\sigma \in \mathcal{F}(X)} f(P_\sigma X)$$

**Theorem 5.3.3** (Invariant Guarantee):
$\bar{f}$ is permutation invariant regardless of $f$.

**Optimal Frame Selection**:
For computational efficiency, choose minimal frames:
$$\mathcal{F}_{\min}(X) = \{\sigma : P_\sigma X \text{ satisfies some canonical criterion}\}$$

```python
def permutation_equivariant_layer(X):
    """
    Input: X ∈ R^{n×d}
    Output: Y ∈ R^{n×d'}
    """
    # Pointwise transformation
    H = MLP(X)  # R^{n×h}
    
    # Aggregate across nodes
    h_global = mean(H, dim=0, keepdim=True)  # R^{1×h}
    
    # Combine local and global
    Y = MLP(concat(H, repeat(h_global, 'h -> n h', n=X.shape[0])))  # R^{n×d'}
    
    return Y
```

**RTT Principle**: Integration of rotational (SO(3)) and permutation (S_n) symmetries.

**Joint Symmetry Group**:
$$G = SO(3) \times S_n$$

**Joint Equivariance**:

## 6. Julia Implementations

### 6.1 Symmetric Group Basics

```julia
"""
Permutation Group S_n implementation
"""
struct Permutation
    mapping::Vector{Int}
    
    function Permutation(perm::Vector{Int})
        n = length(perm)
        @assert sort(perm) == 1:n "Invalid permutation"
        new(perm)
    end
end

# Composition
function Base.:*(σ::Permutation, τ::Permutation)
    n = length(σ.mapping)
    @assert length(τ.mapping) == n
    Permutation([σ.mapping[τ.mapping[i]] for i in 1:n])
end

# Inverse
function Base.inv(σ::Permutation)
    n = length(σ.mapping)
    inv_map = zeros(Int, n)
    for i in 1:n
        inv_map[σ.mapping[i]] = i
    end
    Permutation(inv_map)
end

# Transposition
function transposition(n::Int, i::Int, j::Int)
    perm = collect(1:n)
    perm[i], perm[j] = perm[j], perm[i]
    Permutation(perm)
end

# Cycle decomposition
function cycles(σ::Permutation)
    n = length(σ.mapping)
    visited = falses(n)
    cycles_list = Vector{Vector{Int}}()
    
    for start in 1:n
        if !visited[start]
            cycle = Int[]
            current = start
            while !visited[current]
                push!(cycle, current)
                visited[current] = true
                current = σ.mapping[current]
            end
            push!(cycles_list, cycle)
        end
    end
    return cycles_list
end

# Cycle type (partition)
function cycle_type(σ::Permutation)
    c = cycles(σ)
    sort!([length(cycle) for cycle in c if length(cycle) > 1], rev=true)
end

# Sign of permutation
function Base.sign(σ::Permutation)
    cycles_list = cycles(σ)
    n = length(σ.mapping)
    num_cycles = length(cycles_list)
    # sign = (-1)^(n - number of cycles)
    (-1)^(n - num_cycles)
end
```

```julia
using LinearAlgebra

"""
Generate permutation matrix from permutation
"""
function permutation_matrix(σ::Permutation)
    n = length(σ.mapping)
    P = zeros(Int, n, n)
    for j in 1:n
        P[σ.mapping[j], j] = 1
    end
    P
end

# Action on vectors
function Base.:*(P::Matrix{Int}, v::Vector{T}) where T
    n = length(v)
    @assert size(P) == (n, n)
    [sum(P[i,j] * v[j] for j in 1:n) for i in 1:n]
end

# Tensor power action
function tensor_permutation_action(σ::Permutation, tensor::Array{T,N}) where {T,N}
    # Permute the first N indices according to σ
    # This implements σ · (v₁ ⊗ v₂ ⊗ ... ⊗ v_N)
    perm_indices = [σ.mapping[i] for i in 1:N]
    permute_axes(tensor, perm_indices)
end
```

```julia
Young diagram represented by partition
"""
struct YoungDiagram
    partition::Vector{Int}  # λ = (λ₁ ≥ λ₂ ≥ ... ≥ λₖ)
    
    function YoungDiagram(parts::Vector{Int})
        @assert all(parts[i] >= parts[i+1] for i in 1:length(parts)-1)
        new(parts)
    end
end

# Hook length at position (i, j)
function hook_length(λ::YoungDiagram, i::Int, j::Int)
    # Hook = boxes to right + boxes below + 1
    parts = λ.partition
    if i > length(parts) || j > parts[i]
        return 0
    end
    arm = parts[i] - j  # boxes to right in row i
    leg = count(p -> p >= j, parts) - i  # boxes below in column j
    arm + leg + 1
end

# Number of SYT (Hook Length Formula)
function num_syt(λ::YoungDiagram)
    n = sum(λ.partition)
    hooks = [hook_length(λ, i, j) 
             for i in 1:length(λ.partition) 
             for j in 1:λ.partition[i]]
    factorial(n) ÷ prod(hooks)
end

"""
Standard Young Tableaux generation
"""
function generate_syt(λ::YoungDiagram)
    n = sum(λ.partition)
    tableaux = Vector{Matrix{Int}}()
    
    # Start with empty tableau
    shape = λ.partition
    tableau = zeros(Int, length(shape), maximum(shape))
    
    function fill_tableau(pos::Int, current::Matrix{Int})
        if pos > n
            push!(tableaux, copy(current))
            return
        end
        
        for i in 1:length(shape)
            for j in 1:shape[i]
                if current[i, j] == 0
                    # Check if valid SYT position
                    valid = true
                    # Check above
                    if i > 1 && current[i-1, j] == 0
                        valid = false
                    end
                    # Check left
                    if j > 1 && current[i, j-1] == 0
                        valid = false
                    end
                    # Check increasing property
                    if valid
                        if i > 1 && current[i-1, j] > 0 && current[i-1, j] >= pos
                            valid = false
                        end
                        if j > 1 && current[i, j-1] > 0 && current[i, j-1] >= pos
                            valid = false
                        end
                    end
                    
                    if valid
                        current[i, j] = pos
                        fill_tableau(pos + 1, current)
                        current[i, j] = 0
                    end
                end
            end
        end
    end
    
    fill_tableau(1, tableau)
    return tableaux
end
```

```julia
"""
Symmetric power Sym^n(V)
"""
function symmetric_power(V::Vector{T}, n::Int) where T
    d = length(V)
    # Basis for symmetric power: monomial symmetric polynomials
    # dim = C(d+n-1, n)
    dim = binomial(d + n - 1, n)
    
    result = zeros(T, dim)
    idx = 1
    # Generate multisets of size n from {1,...,d}
    for combo in with_replacement_combinations(1:d, n)
        # Product of corresponding V elements
        result[idx] = prod(V[combo])
        idx += 1
    end
    result
end

Exterior power Λ^n(V)
"""
function exterior_power(V::Vector{T}, n::Int) where T
    d = length(V)
    if n > d

    # Basis for exterior power: wedge products
    # dim = C(d, n)
    dim = binomial(d, n)
    
    result = zeros(T, dim)
    idx = 1

        # Wedge product = determinant of matrix
```

```julia
struct PermEquivariantLayer

    output_mlp::Chain

function PermEquivariantLayer(d_in::Int, d_hidden::Int, d_out::Int)
    PermEquivariantLayer(
        Chain(Dense(d_in, d_hidden, relu), Dense(d_hidden, d_hidden)),
        Chain(Dense(d_hidden + d_hidden, d_hidden, relu), Dense(d_hidden, d_out))
    )
end

function (layer::PermEquivariantLayer)(X::Matrix{Float32})
    # X: n × d_in matrix (n points, d_in features each)
    n, d_in = size(X)
    
    # Pointwise transformation
    H = layer.pointwise_mlp(X')'  # n × d_hidden
    
    # Global aggregation (invariant)
    h_global = mean(H, dims=1)  # 1 × d_hidden
    
    # Broadcast and concatenate
    H_global = repeat(h_global, n, 1)  # n × d_hidden
    H_combined = hcat(H, H_global)  # n × (2 * d_hidden)
    
    # Output transformation
    Y = layer.output_mlp(H_combined')'  # n × d_out
    
    return Y
end

Permutation-invariant layer (for global outputs)
"""
struct PermInvariantLayer
    equivariant::PermEquivariantLayer
    output_mlp::Chain
end

    PermInvariantLayer(
        PermEquivariantLayer(d_in, d_hidden, d_hidden),
        Chain(Dense(d_hidden, d_hidden, relu), Dense(d_hidden, d_out))
    )
end
```

```julia
"""
Frame averaging for permutation-invariant predictions
"""
function frame_average(f::Function, X::Matrix{T}; frame_type::Symbol=:full) where T
    n, d = size(X)
    
    if frame_type == :full
        # Full frame: all permutations (expensive!)
        frames = [permutation_matrix(Permutation(collect(perm))) 
                  for perm in permutations(1:n)]
    elseif frame_type == :canonical
        # Canonical frame: sort-based
        # Choose permutation that makes X sorted by first feature
        order = sortperm(X[:, 1])
        frames = [permutation_matrix(Permutation(order))]
    elseif frame_type == :adjacent
        # Adjacent transpositions only
        frames = [Matrix{T}(I, n, n)]  # identity
        for i in 1:n-1
            push!(frames, permutation_matrix(transposition(n, i, i+1)))
        end
    else
        error("Unknown frame type: $frame_type")
    end
    
    # Average over frame
    result = zero(f(X))
    for P in frames
        result .+= f(P * X)
    end
    result ./= length(frames)
end
```
