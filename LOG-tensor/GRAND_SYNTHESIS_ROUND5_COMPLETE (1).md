## 4. AGENT STATE THEORY

### Core Discovery: All-Zero Equilibrium

```
When agent state = all 0s:
  → System and model MATCH perfectly
  → No computation needed
  → Entropy at maximum under uniform priors

When state ≠ all 0s:
  → MISMATCH detected
  → PERCEPTION (not just sensation)
  → "What should have been" vs "What did happen"
```

### Perception vs Sensation Distinction

| Aspect | Sensation | Perception |
|--------|-----------|------------|
| Input | Raw transduction | Sensation + Comparison |
| Computation | None | Expectation comparison |
| Information | I(s) | I(s) + I(s - E) |
| Consciousness | None | Threshold-dependent |

### Implementation

```python
class AgentStateTheory:
    def __init__(self, dim):
        self.state = np.zeros(dim)  # Start at zero
        self.expectation = None
        self.perception_threshold = 0.1
    
    def update(self, sensation):
        if self.expectation is not None:
            mismatch = np.linalg.norm(sensation - self.expectation)
            if mismatch > self.perception_threshold:
                return self.learn_from_mismatch(sensation, self.expectation)
        return sensation  # Just sensation, no perception
```
