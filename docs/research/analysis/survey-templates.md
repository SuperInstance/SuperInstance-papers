## Interview Protocol (30 Minutes)

### Pre-Interview Setup
**Participants:** Beta users with 10+ sessions
**Incentive:** $50 Amazon gift card
**Tools:** Zoom, screen share, recording consent
**Schedule:** 10 per week, batches of similar backgrounds

### Structured Flow

**1. Introduction (2 min)**
"Hi [Name], thanks for joining! This helps us improve SuperInstance.AI. I'll ask about your experience and show us some features. The recording is internal only. Any questions before we start?"

**2. Discovery Questions (5 min)**
- "Tell me about your current spreadsheet work..."
- "What tools do you use daily?"
- "What's your biggest spreadsheet frustration?"
- "How did you find SuperInstance.AI?"

**3. Live Task Walkthrough (12 min)**
"Let's screen share and show me how you use SuperInstance. You can pick any recent project."

**Prompts During Demo:**
- "What are you trying to accomplish?"
- "Walk me through your thinking..."
- "What would you do without this tool?"
- "What surprised you?"
- "Any confusion points?"

**4. Comparison Questions (4 min)**
- "How does this compare to Excel/Sheets help?"
- "What does Excel do better?"
- "What impressed you most?"
- "What would you change first?"

**5. Future Vision (4 min)**
- "If you were CEO for a day...?"
- "What should we build next?"
- "Who else should see this tool?"
- "What would get you to upgrade to paid?"

**6. Closing (3 min)**
- "Anything else we haven't covered?"
- "May we follow up with future betas?"
- "We'll send your gift card today!"
- "Thank you for your time and insights!"

### Post-Interview
**Immediate Notes:**
- Key quotes (word for word)
- Surprising insights
- Feature requests
- Pain points
- Emotional moments

**Weekly Synthesis:**
- Theme identification
- Quote compilation
- Feature priority ranking
- User segments identified

## Feedback Portal Design

### In-App Widget
**Location:** Bottom-right corner (always visible)
**Options:**
1. 🐛 Report Bug → Form asks for steps, screenshot
2. 💡 Suggest Feature → Vote on existing or add new
3. 📖 Request Tutorial → Template: "Show me how to..."
4. ❓ Ask Question → Links to docs/community
5. 📊 Send Feedback → 3-question survey

### Community Forum Structure
**Categories:**
- **Bug Reports** → Public tracking with labels
- **Feature Requests** → Voting system
- **How Do I...** → Tutorial requests
- **Tips & Tricks** → User sharing
- **Integrations** → API discussions
- **Education** → Teaching resources

**Gamification:**
- Points for helpful answers
- Monthly champions
- Beta tester badges
- Tutorials published perks
