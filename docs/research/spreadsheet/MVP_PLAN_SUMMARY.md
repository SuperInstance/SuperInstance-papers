## 🎁 MVP Features

### P0 (Must-Have)
- ✅ Single-cell agent binding (`=AGENT("task", range)`)

## ⚠️ Key Risks

### Technical
- **Performance** (HIGH): Agents too slow → Progressive enhancement
- **Memory** (HIGH): Browser crashes → Hard limits (50 agents)
- **Compatibility** (MEDIUM): Old browsers → Feature detection

### Business
- **Microsoft** (MEDIUM): Competing feature → Move faster, open source
- **Adoption** (MEDIUM): Users don't get it → Clear onboarding
- **Monetization** (LOW): Can't generate revenue → Freemium, enterprise

### Legal
- **Privacy** (HIGH): Data breach → Local-first, GDPR compliance
- **IP** (LOW): Patent infringement → Prior art search, legal review
