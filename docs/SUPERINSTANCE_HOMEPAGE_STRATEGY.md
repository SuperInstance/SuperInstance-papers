# SuperInstance Homepage Strategy

**Project:** SuperInstance.AI Web Presence
**Date:** 2026-03-14
**Status:** Strategic Planning
**Domain:** superinstance.ai
**Technology:** Cloudflare Pages + Workers + Astro

---

## Executive Summary

SuperInstance.AI serves as the central hub for three interconnected products:
1. **SuperInstance Papers** - 60+ academic white papers on mathematical computation
2. **Lucineer Hardware** - Mask-locked inference chips and educational AI
3. **SpreadsheetMoment** - Visual documentation platform for tensor spreadsheets

The homepage must bridge academic research, practical applications, and educational accessibility while making advanced AI concepts comprehensible to everyone from researchers to 5th graders.

## Mobile Responsive Design

### Breakpoint Strategy

**Desktop (1200px+):**
- Full multi-column layout
- Interactive simulations visible
- Side navigation for docs
- Large paper grid (3-4 columns)

**Tablet (768px - 1199px):**
- 2-column layouts
- Collapsible navigation
- Optimized simulation viewing
- Paper grid (2 columns)

**Mobile (< 768px):**
- Single column stack
- Hamburger menu
- Simplified simulations
- Paper grid (1 column)
- Touch-optimized controls

### Mobile-Specific Features

**Progressive Enhancement:**
- Core content loads first
- Interactive elements load asynchronously
- Optional high-bandwidth features

**Touch Optimization:**
- Larger tap targets (44px minimum)
- Swipe gestures for carousels
- Pinch-to-zoom for visualizations
- Haptic feedback where appropriate

**Performance:**
- Lazy loading for below-fold content
- Optimized images (WebP with fallback)
- Minimal JavaScript for core interactions
- Service worker for offline reading

- Traffic sources

### Custom Events

**Event Tracking:**
```javascript
// Paper engagement
plausible.track('PaperView', { paperId: 'P2', phase: 'Phase1' });
plausible.track('PaperDownload', { paperId: 'P2', format: 'pdf' });

// Simulation usage
plausible.track('SimulationStart', { simulation: 'confidence-cascade' });
plausible.track('SimulationComplete', { simulation: 'confidence-cascade', duration: 120 });

// External link clicks
plausible.track('ExternalLink', { destination: 'github', url: 'https://github.com/SuperInstance' });

// Demo interactions
plausible.track('DemoStart', { platform: 'spreadsheet-moment' });
plausible.track('DemoAction', { action: 'cell-click', confidence: 0.95 });
```
